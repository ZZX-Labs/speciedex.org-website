SpeciedexExplorer

SpeciedexExplorer is the analytical and visual discovery client for the Speciedex ecosystem.

It provides interactive access to species profiles, chain history, taxonomy, geography, ecology, observations, specimens, research, media, conservation, provenance, and network state.

It may run against a local SpeciedexCore node, an institutional node, a trusted SpeciedexNet service, or the public SpeciedexWeb gateway.

It never exposes a Homo sapiens species record because human entry is forbidden at the protocol level.

Purpose

Advanced Interactive Discovery

Built for Deep Exploration

SpeciedexExplorer turns verified chain data into maps, taxonomic trees, record graphs, timelines, tables, comparisons, media views, provenance trails, and analytical workspaces.

Node-Aware Client

Explorer may connect to a local SpeciedexCore node, an institutional node, a trusted remote gateway, or a SpeciedexWeb public endpoint.

Read Without Flattening

The interface distinguishes canonical summaries from source assertions, disputed records, confidence levels, historical states, and unresolved scientific disagreement.

Replay the Record

Users can move through a species chain block by block or event by event to see how knowledge changed over time.

Discovery

Search and Query

Unified Search

Search scientific names, common names, synonyms, identifiers, traits, descriptions, locations, references, media, validators, providers, chains, blocks, and bounties.

Structured Filters

Filter by taxonomy, geography, habitat, time, evidence, source, confidence, review state, chain height, update age, media type, and conservation condition.

Relationship Queries

Ask questions that cross records, such as which pollinators overlap a plant range, which hosts share a parasite, or which threatened species occur within a protected area.

Historical Queries

Reconstruct what the network considered canonical at a selected chain height, release, date, source set, or review state.

Watch the Dataset Evolve

Chain Playback

Play and Pause

Play accepted chain events as a chronological stream, pause on any block, and inspect its complete evidence and validation record.

Speed Control

Replay at stepped rates for teaching, demonstrations, audit work, incident analysis, or accelerated review.

Previous and Next

Move deterministically between adjacent blocks, revisions, assertions, conflicts, and canonical-state changes.

Reverse

Trace the species record backward to see which later statements superseded, qualified, or challenged earlier data.

Repeat

Loop a selected range, taxonomic revision, field campaign, provider import, or conflict-resolution sequence.

Live Mode

Follow validated network activity as new species-chain events propagate through connected nodes.

Scientific Work

Analysis Tools

Distribution Analysis

Measure occurrence density, temporal change, uncertainty, sampling effort, geographic gaps, and introduced or declining range.

Trait Analysis

Compare measurements, life histories, diagnostic traits, developmental stages, variation, and source confidence.

Ecological Networks

Inspect interaction networks and calculate connectedness, dependency, host breadth, or ecosystem roles.

Conservation Analysis

Compare assessments, threats, protection, observation frequency, habitat loss, and evidence age.

Provider Comparison

Compare how sources classify, describe, identify, or assess the same species and expose unresolved differences.

Data Quality

Identify missing fields, stale claims, weak evidence, improbable values, duplicate observations, and candidate bounty tasks.

Trust

Evidence, Provenance, and Citation

Every View Is Traceable

A displayed claim can resolve to its species chain, block or entry, source record, object hash, validator state, and later revisions.

Citation Builder

Generate stable citations for current records, historical chain states, filtered queries, maps, data exports, and analytical notebooks.

Exportable Results

Export permitted views as JSON, JSON Lines, CSV, TSV, GeoJSON, text reports, images, or reproducible query definitions.

No Human Dataset

Explorer does not provide human species pages, human observation search, human biometric analysis, or human tracking functions because the protocol forbids such records.

Deployment

Access Paths

Local Explorer

Connect directly to a user-owned SpeciedexCore node for complete control, privacy, offline work, and local-speed analysis.

Institutional Explorer

Use a university, museum, laboratory, agency, conservation, or archive node with organization-specific collections.

Remote Explorer

Connect to a trusted SpeciedexNet gateway when local storage or node operation is impractical.

SpeciedexWeb

Use the public website for immediate browser access without installing or operating node software.
